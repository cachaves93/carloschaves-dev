SmartApp is an app that aims to provide tools to help brazilian real state investors participate in real state auctions with the required knowledge to make good decisions. Some of its main features includes a personalized real state investment calculator, access to compiled data on the main real state auction opportunities in Brazil, management and notifications for real state auction opportunities over the areas you're interested in. Main tech stacks used were Angular, NodeJS, Ionic(native apps) and Google Cloud Platform.

App na Obra is an app that aims to help civil engineering companies be more structured and organized in its processes. It is currently in development and the main features right now includes tools that provides better management of construction daily reports along with equipment inspection and repairments workflow. Main tools used are Angular, NodeJS, PostgresSQL, Elastic, Google Cloud Platform and Ionic(native apps).
